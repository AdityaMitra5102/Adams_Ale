The smsreader.py class is set to run at startup.

The SIM900A GSM Module is attached to COM17 port.
When this program is runned, the COM port should be
changed accordingly.

SIM900A GSM Module should be connected using RS232
port or using a USB to TTL cable. However, if this is
run on device like a Raspberry Pi, it can be directly
connected to the Rx and Tx ports with jumpers.
